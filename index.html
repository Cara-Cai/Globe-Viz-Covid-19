PROJECT INFO

This project is a 3D Globe visualization of country-wide COVID-19 total cases and total deaths from 2020-01-22 to 2023-03-09.

Considering the GPU capacity and performance of computers, some countries are not indicated on the globe. They are indicated in the table.

The framework and geoJson file used are from Globe GLmodule.The dataset used is parsed data by John Henry Thompson, whose data source is CSSE at Johns Hopkins University. The full project documentation can be accessed Here.

For clarification, the height of the country polygons are mapped based on the total cases of the countries.

This project is contributed to The Covid Impact Project by John Henry Thompson & Shindy Johnson.
